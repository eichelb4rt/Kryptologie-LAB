usage: keygen.py [-h] [-p perm] [-k keylength] in out

positional arguments:
  in            File that the key is read from
  out           File that the generated keys are written to

optional arguments:
  -h, --help    show this help message and exit
  -p perm       File that the permutation is read from (default:
                permutation.txt)
  -k keylength  The length of the key (default: 64)
